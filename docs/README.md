# DUSK – tired yet handsome photo editor

Queue-based portrait editor with real-time preview, batch export, dark theme, and a self-registering modular architecture.

**Features:** crop, smooth, brightness/contrast/saturation/sharpness, vignette, grain, tone (warm/cool), presets, before/split/after view, ZIP import/export, in‑app log viewer.

[Full documentation](docs/)

## Quick start

```bash
pip install Pillow numpy
python main.py

```

## Why the weird name?

Because it's tired (dark UI, handles batch drudgery) yet handsome (it works).
